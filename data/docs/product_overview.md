# CloudSync Platform Overview

CloudSync is a SaaS platform that synchronizes data across multiple cloud storage providers.

## Supported Providers
- Google Drive
- Dropbox
- OneDrive
- Amazon S3

## Pricing Plans
- Free: 5GB storage, 2 providers, basic sync
- Pro ($15/month): 100GB storage, 5 providers, real-time sync
- Enterprise ($50/user/month): Unlimited storage, all providers, SSO, priority support

## Key Features
- Real-time file synchronization
- Conflict resolution (newest wins, manual override, merge)
- File versioning (keep up to 30 versions)
- Team folders with granular permissions
- Activity audit logs
- API access for custom integrations
